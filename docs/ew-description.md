# Toolbar Layout — Never lose your SketchUp workspace again.

**Save, switch and restore every toolbar in one click. Free. No account. No cloud.**

---

## The problem every SketchUp pro knows

You spend an hour arranging your toolbars exactly where you want them — modeling tools on the left, view controls floating top-right, render plugins docked at the bottom. Then SketchUp crashes. Or updates. Or you open a clean install on a new machine. And it's all gone. Drag, dock, repeat — every single time.

**Toolbar Layout** ends that frustration. Forever.

It's a free, lightweight SketchUp extension that remembers exactly where every toolbar lives — and brings them all back in one click. Save once, restore anywhere, switch between workflows instantly. No subscription, no account, no telemetry, no cloud.

---

## Six superpowers in one extension

### Saved Layouts — switch in one click
Name and save any number of toolbar configurations. Go from *Modeling* to *Rendering* to *2D Export* in a single click. Each layout captures every toolbar's position, visibility, floating state, and docking — and survives restarts, updates, crashes, and reinstalls.

### Custom Toolbars — your tools, your way
Mix tools from any plugin into your own custom toolbars. Build a single bar holding the tools you actually use — from native SketchUp, V-Ray, Profile Builder, RoundCorner, ThruPaint, or any other extension — together in one place. Drag, drop, rename, reorder, hide.

### Quick Search — find any tool instantly
Press one keyboard shortcut and find any tool across every extension installed. Real-time fuzzy matching, no menu spelunking. Find what you need in milliseconds — even if you've forgotten which plugin it belongs to. A floating, keyboard-first dialog that stays out of your way.

### Keyboard Shortcut Manager — tame your hotkeys
Assign, edit, override and track shortcuts for any SketchUp command. See your shortcut history, surface conflicts, export your binding set — finally a sane way to manage hotkeys at scale.

### Restore Instantly — one click, exact pixel
One click brings back every toolbar to its exact position after a crash, update, or fresh install. Pixel-perfect on Windows (via native Win32 calls) and reliable on macOS.

### Per-File Layouts — one workspace per project
Optional: attach a layout to a specific `.skp` file. Open the file, your workspace auto-applies. Different projects, different workflows, zero friction.

## Compatibility & install

Works with **SketchUp 2017, 2018, 2019, 2020, 2021, 2022, 2023, 2024 and 2025** — on both **Windows** and **macOS**. No external dependencies, no Ruby gems to install, no admin rights required.

Three steps:

1. Download the `.rbz` from this page.
2. In SketchUp, go to *Extensions → Extension Manager → Install Extension*.
3. Select the `.rbz` file.

That's it. The new **View → Toolbar Layout** menu is your home. Up and running in 60 seconds.
