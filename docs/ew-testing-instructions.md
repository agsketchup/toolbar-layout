After install on SketchUp 2017+ (Windows or macOS), the entry point is the View → Toolbar Layout menu and a small native toolbar with 5 buttons. No license, no account, no activation — fully unlocked and free on install. No external dependencies.

Core flow to test:
1. Arrange and dock toolbars in any positions.
2. View → Toolbar Layout → Save Layout. Name it "Test A".
3. Rearrange toolbars, hide some, dock others. Save as "Test B".
4. Apply → Test A / Test B. Toolbars must snap back to saved positions.

v2.0 features to verify:
• Custom Toolbars — build a bar mixing tools from any plugin.
• Quick Search — floating dialog, fuzzy-find any tool across extensions.
• Keyboard Shortcut Manager — assign, edit, track shortcuts.
• Presets — import / export layouts as .agskp JSON files.
• Per-file layout — attach to a .skp; reopen → layout auto-applies.
