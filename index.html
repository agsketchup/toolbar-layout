Features

A laser-focused extension that does one thing perfectly — and a few more things very well.

Saved Layouts — switch in one click

Name and save any number of toolbar configurations. Go from "Modeling" to "Rendering" to "2D Export" instantly. Each layout captures every toolbar's position, visibility, and floating state — and survives restarts, updates, and crashes.

Save Layout Restore Layout Multiple presets

Custom Toolbars

Group any tools from any plugin into your own toolbars. Mix and match across extensions — one bar for everything you use in a day.

Drag & drop Any plugin

Quick Search — find any tool instantly

Press the search button or keyboard shortcut to find any tool across all your extensions by name. Matching results float to the top.

Installation

Up and running
in 60 seconds.

Three steps. No configuration, no account, no cost.

1

Download the .rbz

Get the extension file from the Extension Warehouse or download directly below.

ags_toolbar_layout.rbz
Extension Warehouse
2

Install in SketchUp

Open SketchUp, then go to:

Extensions
→ Extension Manager
→ Install Extension Select the .rbz file you downloaded
3

Save your layout

Arrange your toolbars exactly as you want them, then:

View
→ Toolbar Layout
→ Save Layout Name it — done. Restore any time.
